Apply

Clicking Apply runs the configuration to disk: writes lynceus.yaml, scaffolds severity_overrides.yaml, imports the bundled watchlist, writes rules.yaml when alerting is enabled, and (under --system scope) chowns the created files to the daemon's system user. Progress streams live to the next page; on failure the completion page offers a Re-run button — the chain is idempotent (atomic writes, idempotent chowns, per-record-dedup'd bundled import), so re-running after fixing an environmental issue is safe. The JSON preview remains available for operators who want to diff the wizard's output against an existing file before applying.
