Welcome to the lynceus-setup wizard

This wizard walks you through the same questions as the interactive CLI flow (lynceus-setup), but in a browser. It validates your answers against the same Config schema the daemon uses, so you cannot produce a configuration the daemon will refuse.

Configuration target: {{ target_path }} ({{ scope }} scope).

Clicking Begin walks you through the form one section at a time. The Review page shows your validated configuration before anything is written. Clicking Apply on Review runs the full write pipeline — writes lynceus.yaml, scaffolds severity_overrides.yaml, imports the bundled watchlist, writes rules.yaml when alerting is enabled, and (under --system scope) chowns the created files to the daemon's system user. Progress streams live to the completion page; if a step fails, the completion page offers a Re-run button (the pipeline is idempotent, so re-running after fixing the underlying issue is safe).
