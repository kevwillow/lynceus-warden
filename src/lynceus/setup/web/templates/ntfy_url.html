Step 7: ntfy broker URL

ntfy is a free push notification service ({{ default_broker }}). Lynceus publishes alerts to a topic you choose; you subscribe to that topic from your phone (ntfy mobile app) or a desktop browser.

Two prompts follow:

  1. ntfy broker URL (this step) — the default ({{ default_broker }}) is the public service. Set this to your own URL if you self-host.
  2. ntfy topic (next step) — a name you pick. Anyone who knows the topic can read AND publish to it, so treat it like a password.

To skip ntfy entirely (no push notifications), leave the URL field empty. You can wire it up later with lynceus-setup --reconfigure.
